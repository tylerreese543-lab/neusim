# neusim

This is a project I made that is designed to simulate a "somewhat" complex environment for my neural networks and to show the synergy between physical and intellectual genetics (even though they are the same thing).

Each organism in the world has functions and can eat food.
Each organism can reproduce and create an organism that is similar but slightly varied from the original.

As organisms reproduce, variations that are superior are favored and thus live on. Defective ones do not get to pass and die as a result.

Simulating this shows the emergent behaviors- even some shown in nature itself.
